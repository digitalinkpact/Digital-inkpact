# Digital-inkpact — Autonomous Bid Processing Assistant

A fully local, zero-cost AI assistant for construction bid processing. See [`ai-assistant/README.md`](ai-assistant/README.md) for complete documentation.

## Quick Start

```bash
cd ai-assistant
bash install.sh        # macOS/Linux
# or
.\install.ps1          # Windows PowerShell

bash scripts/run.sh    # Start all services
# Open http://localhost:3000
```
